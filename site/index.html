HTML

HTML - Язык разметки веб-страницы
<html>..</html>
Самый главный тег, содержит все остальные html-теги.
<head>..</head>
Содержимое этого тега не отображается на сайте. Он нужен для предоставления дополнительной информации браузеру(например: название сайта и кодировка сайта)
<body>..</body>
Содержит “тело” нашего сайта(в нем содержится весь контент нашей веб страницы)
<title>..</title>
Название веб-страницы
<style>..</style>
Cодержит внутри CSS стили
<link>
Подключение внешних ресурсов(CSS стили, шрифты и т.п.)
<p>..</p>
Содержит текстовую информацию, отображаемую на сайте.
<img src=”img.jpeg”>
Отображает картинку на сайт.Имеет обязательный атрибут в котором указывается изображение
<h1>..</h1>
Заголовок
<br>
Переход на новую строчку
<a href="[адрес]">..</a>
Ссылка на другую страницу. Имеет обязательный атрибут, в котором указывается ссылка
<div>..</div>
Универсальный тег, может содержать в себе другие теги. Он очень широко используется в разработке веб-страниц для деления страницы на секции и группирование элементов
Класс - это произвольное имя HTML-элементов, которое мы можем изменять с помощью CSS-стилей Пример:
<div class="myClass">..</div>


CSS

CSS - Язык таблиц стилей(например цвет, шрифт и тень), который прикрепляется к HTML-элементам
Структура CSS:
[элемент] {
[свойство]: [значение];
[свойство]: [значение];
}
Что бы изменить эл. с классом, нужно использовать точку:
.myClass{
color: red;
font-family: cursive;
}
Пример(Изменение цвета и шрифта текста):
p{
color: red;
font-family: cursive;
}
Цвета в CSS задаются одним из трех способов:
1. Название цвета: Skyblue
2. RGB-палитра: rgb(135,206,235)
3. Шестнадцатеричный код: #87CEEB
:hover
Псевдокласс, добавляет стиль к элементу при наведении на него курсора мыши, указывается после имени элемента.
:active
Псевдокласс, добавляет стиль к элементу при нажатие по нему курсора мыши, указывается после имени элемента.
